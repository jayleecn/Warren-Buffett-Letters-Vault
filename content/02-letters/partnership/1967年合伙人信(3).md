# 巴菲特合伙人信 — 1967年

巴菲特合伙基金有限公司
基威特大厦610号
内布拉斯加州奥马哈 68131
电话 042-4110

[[沃伦·巴菲特|沃伦·E·巴菲特]]，普通合伙人
威廉·斯科特
约翰·M·哈丁

1967年11月1日

致我的合伙人：

随信附上：

1. 两份1968年承诺书，一份由你保留，一份寄回给我们。你可以在12月31日之前随时修改承诺书，所以请尽早寄回，如需更改，来信或致电告知即可。承诺书在12月31日绝对定稿——此规则不容例外，因为届时我会将其移交给毕马威会计师事务所。

2. 一封税务信函，提供我目前对1967年已实现投资交易的年终预估。今年我们处置了一些重要持仓，在此过程中实现了约2,400万美元的收益。因此，在寄回承诺书之前，请务必仔细审视你的税务状况，以免明年四月让国税局感到失望。如需帮助，约翰·哈丁很乐意帮你粗略估算税务需求。

3. 一份2月2日寄给你的信函副本，其中包含税务信中解释的计算所需数据。过去我们总让你翻箱倒柜找这封信——一位合伙人提出了绝妙的建议，让我们此时随信附上一份副本。

4. 一份"基本规则"副本，其中第四条和第五条已完全修改，以反映我10月8日信中解释的新条件。如果第四条中设定的目标即使实现也不能令你满意（而且那只是目标，不是承诺），你最好在年底撤资。在目前的条件下，部分合伙人很可能有更好的资金去处，如果确实如此，做出调整是合理的。

任何撤资将在1月5日之前支付。你可以提取100美元到全部权益之间的任何金额。同样，追加投资金额不限，应在1月10日之前到账。如果追加资金在11月到账，将被视为预付款，到年底前按6%计息。12月到账的追加资金不计利息。

在我写这封信时，[[道琼斯工业指数]]大约在880点，今年以来的整体回报约为正15%。我们的业绩约为正27%，折算为有限合伙人约正22%（扣除每月提款前）。这不涉及我们对年末[[控制类投资]]估值相对1966年底的任何变化。在12月22日确认你承诺书状态的信中，我将告知你我们对[[多元零售公司]]和[[伯克希尔·哈撒韦]]的估值，并附上与该估值相关的财务资料。这一调整可能会为我们今年的业绩增加少量收益。

在解读我们今年剩余时间乃至明年的业绩时，你应当了解：我们不仅有约2,000万美元投资于[[控制类投资|控制类公司]]，还有超过1,600万美元的短期国债。目前合计超过3,600万美元，这些资金显然不会参与股市可能出现的任何上涨。这并不反映我对市场的判断；仅仅意味着我找不到任何明显有利可图且安全的投资去处（从长期[[内在价值]]角度看，而非短期报价角度）。我们通常在每年初都有几个即将孵化的投资机会；但目前巢穴几乎是空的。这种情况可能会很快改变，也可能持续一段时间。提及此事只是为了在近期股市走强的情况下，适度调低你对我们业绩的预期。

此致敬意，

沃伦·E·巴菲特

WEB/eh
