# 巴菲特致股东信 — 核心术语表

本术语表确保 91 篇信件翻译中术语的一致性。

## 投资概念

| 英文 | 中文译名 | 说明 |
|------|---------|------|
| Intrinsic Value | 内在价值 | 企业真实价值的估算 |
| Margin of Safety | 安全边际 | 买入价格与内在价值之间的差距 |
| Moat / Economic Moat | 护城河 / 经济护城河 | 企业的持久竞争优势 |
| Compounding | 复利 | 收益再投资产生的增长效应 |
| Float | 浮存金 | 保险业务中先收后赔的资金 |
| Circle of Competence | 能力圈 | 投资者真正理解的领域 |
| Mr. Market | 市场先生 | 格雷厄姆的市场拟人化比喻 |
| Owner Earnings | 所有者盈余 | 巴菲特定义的真实盈利指标 |
| Capital Allocation | 资本配置 | 资金在不同用途间的分配 |
| Book Value | 账面价值 | 资产减负债后的净值 |
| Look-through Earnings | 透视盈余 | 包含被投公司未分配利润的盈余 |
| Franchise | 特许经营权 | 具有定价权和高回报的商业模式 |
| Goodwill | 商誉 | 收购价超过净资产的部分 |
| Retained Earnings | 留存收益 | 未作为股息分配的利润 |
| Opportunity Cost | 机会成本 | 选择一项投资而放弃其他选择的代价 |
| Skin in the Game | 利益一致 | 管理层与股东利益绑定 |
| Wonderful Company at Fair Price | 以合理价格买入好公司 | 芒格影响下的投资转变 |
| Cigar Butt Investing | 捡烟蒂投资 | 格雷厄姆式的低价买入策略 |
| Competitive Advantage | 竞争优势 | 企业优于对手的持久能力 |
| Durable Competitive Advantage | 持久竞争优势 | 长期可维持的竞争优势 |

## 投资分类

| 英文 | 中文译名 | 说明 |
|------|---------|------|
| Generals | 低估类投资 / 一般性投资 | 被低估的普通股票投资 |
| Workouts | 套利类投资 | 依赖特定公司事件获利的投资 |
| Controls | 控制类投资 | 取得控制权的投资 |
| General Issues | 大盘跟随型投资 | 随大盘波动的投资 |

## 保险与财务术语

| 英文 | 中文译名 | 说明 |
|------|---------|------|
| Underwriting Profit/Loss | 承保利润/亏损 | 保费收入减去赔付和费用 |
| Combined Ratio | 综合成本率 | 赔付率加费用率 |
| Premium Volume | 保费收入 | 收取的保险费总额 |
| Net Worth | 净资产 | 总资产减总负债 |
| Per-share Book Value | 每股账面价值 | 净资产除以总股数 |
| Operating Earnings | 经营利润 | 日常经营产生的利润 |
| Unrealized Gains | 未实现收益 | 持有资产升值但未卖出的收益 |
| Deferred Tax | 递延税款 | 延迟缴纳的税款 |
